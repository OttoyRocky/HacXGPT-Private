# 📜 Código de Conducta para HacXGPT

## Nuestro Compromiso
Crear un ambiente abierto, respetuoso y profesional para todos los colaboradores, independientemente de su experiencia, identidad o background.

## Comportamiento Esperado
✅ Usar lenguaje inclusivo y respetuoso
✅ Aceptar críticas constructivas profesionalmente
✅ Enfocarse en lo mejor para la comunidad
✅ Mostrar empatía hacia otros miembros
✅ Respetar las leyes y regulaciones aplicables

## Comportamiento Inaceptable
❌ Comentarios sexualizados o violencia
❌ Trolling, insultos o ataques personales
❌ Publicar información privada de terceros
❌ Acosar a otros miembros
❌ Promover actividades ilegales

## Aplicación
Los maintainers se reservan el derecho de:
- Eliminar comentarios inapropiados
- Bloquear usuarios temporalmente
- Banear permanentemente a infractores

## Reportes
Reporta incidentes a: conducta@hacxgpt.local (o crea un issue privado)
Todos los reportes serán tratados con confidencialidad.
