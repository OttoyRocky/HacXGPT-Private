# Guía de Uso - HacXGPT v7.0

## Menú Principal
1. **Reconocimiento Básico** - WHOIS, DNS, subdominios
2. **Escaneo Web** - Whatweb, Nikto, directorios
3. **Análisis de Red** - Nmap, ping, traceroute
4. **Suite de Pentesting** - Metasploit, sqlmap, hydra
5. **Generar Reportes** - HTML, texto, plantillas
6. **Herramientas Avanzadas** - Malware, forense, OSINT
7. **Chat Libre** - Preguntas técnicas con respuestas detalladas
8. **Escaneo Sigiloso** - Técnicas avanzadas de Nmap
