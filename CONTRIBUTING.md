# 🤝 Guía para Contribuir a HacXGPT

## 📋 Proceso de Contribución
1. **Fork** el repositorio
2. Crea tu rama: `git checkout -b feature/nueva-funcionalidad`
3. **Commit** cambios: `git commit -m 'Añade: nueva funcionalidad X'`
4. **Push** a la rama: `git push origin feature/nueva-funcionalidad`
5. Abre un **Pull Request** en GitHub

## ✅ Requisitos para Pull Requests
- [ ] Código comentado en español/inglés
- [ ] Sin credenciales ni datos sensibles
- [ ] Funcionalidad probada en entorno controlado
- [ ] Documentación actualizada
- [ ] Disclaimer legal incluido en nuevas funciones

## 🛡️ Política de Seguridad
**NO** subas vulnerabilidades a issues públicos.
Reporta a través de SECURITY.md (crea un issue privado o email)

## 📝 Estilo de Código
- Usar nombres descriptivos para variables
- Comentar funciones complejas
- Seguir el formato existente (indentación, colores, etc.)
- Incluir mensajes de error claros para el usuario

## ⚖️ Aspectos Legales
Al contribuir, aceptas que tu código:
- Será bajo licencia MIT
- Es SOLO para fines educativos
- No debe facilitar actividades ilegales
- Incluirá advertencias de uso responsable

## ❓ Preguntas
Si tienes dudas, abre un issue con la etiqueta "pregunta"
