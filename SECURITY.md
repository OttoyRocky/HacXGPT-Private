# Política de Seguridad

## Reporte de Vulnerabilidades
Si encuentras una vulnerabilidad en este código:
1. NO la compartas públicamente
2. Reporta de forma privada al mantenedor
3. Espera parches antes de divulgar

## Uso Responsable
Este software es solo para:
- Entornos de laboratorio controlados
- Pruebas en sistemas propios
- Educación en seguridad informática

## Datos Sensibles
No incluir en commits:
- Direcciones IP reales
- Credenciales de acceso
- Tokens de API
- Información personal
